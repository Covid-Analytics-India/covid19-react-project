## Setup

```
npm i && npm start
```

## Configure backend server

Edit .env file to change backend server and do npm start.

## Linting

### ESLint

To identify problematic pattern in JavaScript code

```
npm run eslint
```

To fix problematic pattern in JavaScript code

```
npm run eslint:fix
```

### Prettier

code formatter

```
npm run format
```

### Sass-lint

To auto fix scss/css

```
npm run scsslint
```
